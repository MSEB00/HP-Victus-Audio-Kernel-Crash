# Security Policy

## Supported Versions

This repository is an engineering investigation and does not distribute executable software.

## Reporting Security Vulnerabilities

This repository is **not** intended for reporting security vulnerabilities affecting:

* HP
* AMD
* Microsoft
* Riot Games
* NVIDIA
* Realtek

If you believe you have discovered a genuine security vulnerability, please report it directly to the affected vendor using their official security disclosure program.

## Repository Scope

This project focuses exclusively on:

* Kernel crash analysis
* Driver investigation
* Audio subsystem behavior
* Reproducible system crashes
* Engineering documentation

Security reports unrelated to this investigation may be closed.
