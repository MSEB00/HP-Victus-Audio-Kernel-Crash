# Contributing

Thank you for taking the time to contribute to this investigation.

This repository documents a reproducible Windows kernel crash involving audio endpoint transitions on HP Victus laptops. Contributions from the community are welcome and greatly appreciated.

## Ways to Contribute

You can contribute by:

* Reporting reproducible crashes
* Testing reproduction steps
* Sharing WinDbg analysis
* Uploading Reliability Monitor screenshots
* Providing Event Viewer logs
* Comparing driver versions
* Testing BIOS or driver updates
* Suggesting possible root causes
* Submitting confirmed workarounds or fixes

## Before Opening an Issue

Please:

* Search existing issues first.
* Use the appropriate issue template.
* Include as much technical information as possible.
* Avoid duplicate reports.

## Recommended Information

Include whenever possible:

* Laptop model
* CPU and GPU
* Windows build
* BIOS version
* Realtek driver version
* AMD ACP driver version
* Reproduction steps
* Reliability Monitor screenshots
* Event Viewer logs
* WinDbg output
* Crash dump (if available)

## Crash Dumps

Do not upload extremely large memory dumps directly to GitHub.

Instead:

* Compress the dump.
* Upload it to a cloud service (Google Drive, OneDrive, Dropbox, etc.).
* Share a public download link.

## Investigation Guidelines

Please distinguish between:

* Confirmed observations
* Experimental findings
* Personal hypotheses

Whenever possible, support conclusions with evidence.

## Code of Conduct

Please remain respectful and constructive.

The goal of this repository is collaborative engineering and technical investigation.
